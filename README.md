# project_gutenberg
網路爬蟲 - 古騰堡計畫 抓取書名含中文的書籍

## 內容摘要
爬取中文書籍，共 477 本。
耗時 : 約 5分鐘

## 安裝套件
- requests 2.32.2
- beautifulsoup4 4.12.3
- selenium 4.27.1
